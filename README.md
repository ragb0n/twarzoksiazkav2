## General info
Project of simple social media site, created using Symfony 6.3 framework. Twarzoksiążka V2 is a remake of old twarzoksiążka, which was made using pure HTML, CSS, JavaScript and PHP.

## Used technologies
Project is created with:
* Symfony 6.3
* HTML
* CSS
* PHP
* JavaScript
